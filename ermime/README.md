# ermine - 闪电貂

## 简介：基于原子开发构思实现的微型框架

### 原子开发构思

1. 原子要实现统一的输入输出方法；
2. 通过不同的原子的完全体相互组合（包括功能的匹配、数据的传递
和样式的组装）最终构成系统；
3. 在不同的系统间，原子的完全体定义可以不同。比如：在 system A
 中原子 ⍴ 进化一次成为完全体。在 system B 中原子 ⍴ 进化两到三
次成为完全体;
